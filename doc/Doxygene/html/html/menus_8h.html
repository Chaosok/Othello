int * quit,

TTF_Font * font1,
TTF_Font * font2 )

Gère les événements SDL pour le menu "Rejoindre une partie réseau".

Cette fonction intercepte les événements clavier et texte pour permettre à l'utilisateur :

  • de saisir une adresse IP,
  • de se connecter à un serveur

En cas de validation (Entrée), elle tente de se connecter à l'IP saisie, puis lance le jeu si la connexion est réussie.

Paramètres
ePointeur vers l'événement SDL courant.
quitPointeur vers le drapeau de sortie du programme.
rendererPointeur vers le renderer SDL utilisé pour lancer le jeu après connexion.
font1Police utilisée dans le jeu.
font2Deuxième police utilisée dans le jeu.
